# Sreda – комплексний курсовий проєкт

Тема роботи: **Програмна система мовного інтерфейсу на основі методів штучного інтелекту**

Автор: **Пивовар Станіслав Євгенович**
Група: **ПЗПІ-23-9**
Спеціальність: **121 – Інженерія програмного забезпечення**

## Опис проєкту

Sreda — це програмна система мовного інтерфейсу, яка забезпечує обробку голосових і текстових команд користувача, визначення наміру, виконання демонстраційних дій, роботу з віртуальними пристроями, журналювання подій та відображення результатів у користувацькому інтерфейсі.

## Примітка щодо STT/TTS

Для роботи мовного інтерфейсу локально використовуються модулі STT/TTS, зокрема Whisper.cpp, Piper TTS та FFmpeg. Через великий розмір моделей STT/TTS вони не входять до репозиторію та встановлюються локально відповідно до налаштувань середовища.
